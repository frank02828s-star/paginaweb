MIGRACIÓN INCLUIDA EN ESTA VERSIÓN

1. Backend base en PHP + MariaDB:
   - /src/bootstrap.php
   - /src/Support/*
   - /auth/*.php
   - /database/schema.sql

2. Protección de secretos:
   - Las credenciales NO van hardcodeadas.
   - Debes crear variables de entorno reales usando .env.example como guía.
   - APP_KEY es obligatoria para cifrar correo y cédula.

3. Datos sensibles protegidos:
   - password_hash() para contraseñas
   - correo cifrado + hash ciego para búsqueda
   - cédula cifrada + hash ciego para búsqueda

4. Registro de usuarios:
   - desde el modal dinámico del header
   - automáticamente desde checkout al confirmar la compra

5. Panel de usuario:
   - /user/index.php
   - header con menú desplegable:
     * Mi cuenta
     * Cerrar sesión

6. Inventario:
   - NO se migró todavía, según tu instrucción.

IMPORTANTE
- Para que el envío de correos funcione de forma robusta, debes configurar el servidor de correo en el hosting.
- Si quieres, en el siguiente paso puedo migrar:
  a) órdenes reales a DB
  b) checkout completo persistido en DB
  c) inventario a MariaDB
  d) panel admin
